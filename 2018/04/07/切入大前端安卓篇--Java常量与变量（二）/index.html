整型字面值

Java中有三种表示整数的方法：十进制、八进制、十六进制。

  • 八进制：以0开头，包括0-7数字。如037，056
  • 十六进制：以0x或0X开头，包括0-9即其A到F，如0xabcf。
  • 0xabcfL，是一个长整型。

浮点型字面值

  • 默认情况表示double类型。例如：123.4表示的是一个单精度浮点型
  • 123.4d或123.4D表示双精度浮点型。
  • 123.4f或123.4F表示单精度浮点型。
  • 双精度浮点型比单精度浮点型范围大，所以将双精度浮点型转为单精度浮点型会发生精度丢失。

数据类型存储

数据类型按照作用范围分为：类级、对象实例级、方法级、块级。方法级内的变量可以称为局部变量。

  1. 内存：Java中内存分为栈、堆、常量池。
  2. 方法级内的局部变量在栈中如何存取？

字符型字面值

  • 字符型字面值使用单引号的单个字符表达。

  • ASICII码

  1. 是一个约定的规则，保证了通信传达信息的一致性。
  2. 7位二进制数 - 标准ASICII码
  3. 8位二进制数 - 扩展ASICII码
  4. 总而言之，char ch = 65 表示的是A,即利用ASCII码表，使得整型对应一个字符，即整型和字符互相转换。
  • char类型是2个字节，即16个二进制位，数据范围是0~65535。char类型代表的无符号整数位。

unicode编码

称为统一码，万国码。目标是支持世界上所有的字符集。

布尔类型字面值

字符串字面值

字符串不属于基本数据类型，它是类。是双引号引起来的0个或多个字符。
