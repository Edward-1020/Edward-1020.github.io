内存 栈 常量池 堆

  • 内存空间中。栈存放类的引用，常量池存放字符串常量，堆存放字符串对象。
  • str1和str2的引用均指向常量池中的imooc，str3指向堆中的imooc的字符串对象。

String不可变性

  • String对象一旦创建，便不可修改。修改String对象，其实是创建了新的对象，所指向的内存空间不变。
  • 字符串的不可变性带来副作用。如果要执行复杂的字符串操作，会生成大量的中间变量，消耗过的内存。

StringBuilder解决不可变性

  • StringBuilder不具有不可变性，当频繁操作字符串，使用StringBuilder。
  • 类似的还有StringBuffer。
  1. 二者相似。
  2. StringBuffer是线程安全的，StringBuilder则没有，StringBuilder性能略高。
  3. 大部分的字符串处理是单线程，所以一般使用StringBuilder。
