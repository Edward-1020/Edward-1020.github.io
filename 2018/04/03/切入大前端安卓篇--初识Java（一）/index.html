Java简介

JVM

  • JVM，Java虚拟机
  • JVM是实现Java跨平台的关键

Java程序执行流程

  1. program.java的源文件。
  2. 经过编译器compiler，编译为Program.class的字节码文件
  3. 字节码文件交由解释器interpreter执行，JVM虚拟机解释执行Java编译而成字节码文件，将字节码文件解释成具体机器上的机器码这样的二进制指令。
  4. 这样，JVM保证Java可以一次编译，到处执行。

JDK

JDK是Java语言的软件开发工具包。

JDK中的两个主要组件

  • javac -编译器，将源程序转成字节码
  • java 运行编译后的java程序

JRE

  • JRE是Java Runtime Environment，Java运行环境
  • 包括Java虚拟机（JVM），Java核心类库和支持文件

JRE和JDK区别

  • JRE负责运行Java环境。
  • JDK面向开发人员，JDK附带JRE。

JRE、JDK、JVM关系

  1. JDK = JRE + JVM
  2. JDK = JRE + 开发工具集
  3. JRE = JVM + JavaSE标准类库

Java平台

  • JavaSE Java标准版，一般用于开发桌面程序
  • JavaEE JavaSE为JavaEE提供基础，是Java企业版，开发Web程序。
  • JavaME 开发移动设备。
